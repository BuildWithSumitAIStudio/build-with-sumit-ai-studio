AI Excel Data Analyzer (Groq + Python) :-

Automatically analyze Excel data, generate charts, and extract business insights using AI.
This project uses Groq LLMs to intelligently understand datasets, create visualization code, and deliver meaningful insights — just like a real data analyst 🤯

Steps :-

Load Excel data using Pandas
Generate structured dataset context
AI generates visualization code dynamically
Execute charts automatically using Matplotlib
Generate business insights using AI

Architecture :-

Excel → Data Processing → Context Generation → AI Code → Chart Execution → Insights

Tech Stack :-

Python
Pandas
Matplotlib
Groq API (LLM)
